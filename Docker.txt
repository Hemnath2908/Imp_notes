Docker commands :
	To list out docker images -> docker image ls
	To pull docker image -> docker pull <ImageName> -> docker pull openjdk:17
	To build docker image -> docker build -t <ImageName> . -> docker build -t app.jar .
	To Run docker image -> docker run -p 9090:8080 app.jar
